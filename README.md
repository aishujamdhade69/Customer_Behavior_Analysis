# Customer_Behavior_Analysis
# 📊 Data Analysis Project

A complete data analysis project involving data processing, exploration, and visualization to generate meaningful insights.

---

## 📌 About

This project focuses on analyzing a dataset using Python, performing Exploratory Data Analysis (EDA), cleaning data, executing SQL queries in MySQL, and creating an interactive Power BI dashboard for visualization.

---

## 🛠️ Tech Stack

* **Python** (Pandas, NumPy, Matplotlib)
* **MySQL** (SQL Queries)
* **Power BI** (Dashboard & Visualization)

---

## ▶️ Workflow

1. Load dataset using Python
2. Perform data cleaning and preprocessing
3. Conduct Exploratory Data Analysis (EDA)
4. Run SQL queries in MySQL for insights
5. Build an interactive Power BI dashboard

---

## 📊 Features

* Data cleaning and preprocessing
* Insightful EDA with visualizations
* SQL-based data analysis
* Interactive dashboard for decision-making
